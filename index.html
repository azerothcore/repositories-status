# Repository status

This simple tool aims to use the GitHub API to bring the organization's repositories and have a preview of their status. Being able to know how many pull requests are open, as well as the issues and other data.

Each of the repositories has a link to it, as well as to the issues and pull requests. The default search is 100. We hope they can be sorted in the future by the number of open pull requests or issues.
